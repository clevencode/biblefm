# bibleco

## Description

Écoutez la Parole de Dieu en direct, à tout moment. Application web : radio en direct dans le navigateur, lecteur audio natif et minuteur de veille.

## Langages

- **Dart** — application Flutter (UI et logique)
- **HTML** / **CSS** — page d’amorçage Web (`web/index.html`)
- **JavaScript** — analytics (`web/analytics.js`)
